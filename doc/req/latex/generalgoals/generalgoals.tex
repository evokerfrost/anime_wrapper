\subsection{General Goals}
\begin{itemize}
    \item A system that gathers data about anime.
    \item A system that can handle data from multiple sources.
    \item A system that interfaces multiple API:s to gather user data.
    \item The system shall be able to be used as a library for further development needs.
    \item The system has defined functions that can be used by external programs to get data in the format specified by this system.
    \item \textit{The system shall be able to handle a predefined set of operations (add, update, delete, etc) for all sources this system interfaces.}
\end{itemize}
